#Garage App
Web app that allows users to post their parking spot and connect with users who would like to rent the spot

##Installation
```
npm install
```

##Running the App

```
node garageapp.js
```

##Tests
Postman tests are located in the /Tests directory

[Postman](https://www.postman.com/)
